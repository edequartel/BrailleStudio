You are working in a Blockly-based project called BrailleStudio.

Task:
Create Blockly blocks and JavaScript generator/runtime code that uses an existing PHP JSON API for audio file selection.

Important working style:
- First make a short implementation plan.
- Then implement the code.
- Keep the code simple, readable, and production-friendly.
- Do not overengineer.
- Prefer explicit block behavior over hidden magic.
- Preserve backwards compatibility where possible.

Project context:
- This is an educational braille learning system.
- End users are teachers and students.

- The backend API already exists and must be used as-is.
- The frontend code should be plain JavaScript.
- The result should be easy to integrate into an existing Blockly app.

Existing API:
Base URL:
https://tastenbraille.com/api/list.php

The API returns JSON like:
[
  {
    "word": "lamp",
    "url": "https://www.tastenbraille.com/braillestudio/sounds/nl/speech/lamp.mp3"
  }
]

Supported folders:
- woorden
- letters
- instructies
- beloningen
- story

API parameters:
- folder            required
- letters           optional, comma-separated first-letter filter
- klanken           optional, comma-separated list; word must contain at least one
- onlyletters       optional, comma-separated list; word may only contain these letters
- onlyklanken       optional, comma-separated list; word may only contain these phonics
- onlycombo         optional true/false; strict combination of onlyletters + onlyklanken
- maxlength         optional integer
- length            optional integer
- limit             optional integer
- randomlimit       optional integer
- sort              optional: asc | desc | random

Example API calls:
- https://tastenbraille.com/api/list.php?folder=woorden
- https://tastenbraille.com/api/list.php?folder=woorden&letters=a,b,k,l
- https://tastenbraille.com/api/list.php?folder=woorden&klanken=a,e,i,ou,ei
- https://tastenbraille.com/api/list.php?folder=woorden&onlyletters=a,l,p,m
- https://tastenbraille.com/api/list.php?folder=woorden&maxlength=4
- https://tastenbraille.com/api/list.php?folder=woorden&sort=asc&limit=10
